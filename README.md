## 💡 주요 기능

### 1. D-Day 확인
연인으로 등록하면 첫 연애일로부터 얼마나 흘렀는지 쉽게 확인할 수 있습니다.
앱 내에서만이 아니라 위젯을 통해서도 확인할 수 있습니다.

### 2. 일정 생성
서로 공유되는 캘린더를 통해 손쉽게 일정을 공유할 수 있습니다.
다른 사람이 작성한 일정은 삭제할 수 없어요!

### 3. 다이어리 작성
서로 공유가 되는 다이어리를 작성하여 추억을 저장할 수 있습니다.
다른 사람이 작성한 다이어리는 건들면 안되겠죠?
